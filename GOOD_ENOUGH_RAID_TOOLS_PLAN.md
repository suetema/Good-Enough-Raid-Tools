# Good Enough Raid Tools V1 Plan

## Scope

Build a TBC Classic Anniversary addon named `Good Enough Raid Tools` with a compact roster matrix, local scan cache, privacy-aware reporting, and whisper-managed public mention opt-outs.

## Milestones

1. Bootstrap addon structure, SavedVariables, slash commands, and movable matrix window.
2. Implement roster scan logic and category detection using visible buffs only.
3. Implement privacy storage, whisper parsing, sanitized reporting, and documentation.
4. Run static verification, fix issues, then commit and push final V1 state.
